ภารกิจที่ 9 - ใส่สี
	น้องๆ จะได้ลองใช้ css ตกแต่งหน้าเว็บ อย่างแรกคือการใส่สีตัวอักษร
	เริ่มภารกิจที่ 9 ได้
- เปิดไฟล์ mission9.html ที่พี่ๆ เตรียมไว้ให้ ด้วย text editor
- ลองสังเกตดู ที่แท็ก head พี่ได้เติมแท็ก style ไว้ให้น้องๆ ใส่ css แล้ว
- ให้น้องพิมพ์ css ต่อไปนี้ลงไประหว่าง tag <style></style>
	#aaa{
		color: red;
	}
- เปิดไฟล์ด้วย browser เพื่อดูสิ่งที่เกิดขึ้น
- ลองเปลี่ยน #aaa เป็น .ccc ดูสิ่งที่เกิดขึ้น
- ลองเปลียน .ccc เป็น p ดูสิ่งที่เกิดขึ้น
